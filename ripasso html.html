cos'è html?

html (hyper text markup language) è un metalinguaggio utilizzato per formattare e impaginare pagine web collegate tra loro attraverso dei link. per modificare lo stile delle pagine è possibile utilizzare CSS, ovvero fogli di stile, questi possono essere:
  • in linea
  • incorporati
  • linkati ad un file esterno
